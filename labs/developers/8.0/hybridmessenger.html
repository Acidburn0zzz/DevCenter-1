Lab series highlights

  • Mobile application development from scratch
  • Uses Ionic v2 beta with Typescript
  • Development using CLI and MS Visual Studio Code
  • User authentication
  • Cloudant database integration
  • Push notifications messaging
  • JSONStore
  • Uses Google Maps Distance Matrix API
  • Uses IBM Bluemix Mobile Foundation server instance
  • Uses IBM Mobile Quality Assurance

Total labs: 12

Time to complete: up to 8.5 hours

Pre-requisites:

Follow the 8.0 Labs required setup to install Ubuntu image, install the MobileFirst Foundation devkit and CLI and register at Bluemix

Make sure you updated your VM image at least to version 3.0.1
