Local Installation

Prerequisites

  • Download Java 7 or 8 JDK
  • Set the JAVA_HOME environment variable to your JDK directory. For example:
    • Windows: C:\Program Files\Java\jdk1.7.0_60
    • OS X: /Library/Java/JavaVirtualMachines/jdk1.7.0_67.jdk/Contents/Home
  • To use the MobileFirst Developer CLI the NPM package manager is required to be installed on the developer workstation.

Installation steps

  1. Go through the installation steps of the Installer:

  2. Start the server:

    • From a Command-line window, navigate to the extracted folder and run the run.sh|cmd command
