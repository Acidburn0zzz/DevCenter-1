    ibm-mobile-first

    Getting started

    $ npm install ibm-mobile-first --save

    Mostly automatic installation

    $ react-native link ibm-mobile-first

    Manual installation

    iOS

    1. In XCode, in the project navigator, right click Libraries ➜ Add Files to [your project's name]
    2. Go to node_modules ➜ ibm-mobile-first and add RNIBMMobileFirst.xcodeproj
    3. In XCode, in the project navigator, select your project. Add libRNIBMMobileFirst.a to your project's Build Phases ➜ Link Binary With Libraries
    4. Run your project (Cmd+R)<

    Android

    1. Open up android/app/src/main/java/[...]/MainActivity.java
      • Add import com.ibm.mobilefirst.RNIBMMobileFirstPackage; to the imports at the top of the file
      • Add new RNIBMMobileFirstPackage() to the list returned by the getPackages() method
    2. Append the following lines to android/settings.gradle:
         include ':ibm-mobile-first'
   project(':ibm-mobile-first').projectDir = new File(rootProject.projectDir,     '../node_modules/ibm-mobile-first/android')
    3. Insert the following lines inside the dependencies block in android/app/build.gradle:
         compile project(':ibm-mobile-first')
