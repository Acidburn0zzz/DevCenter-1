IBM MobileFirst Platform Foundation provides a Native SDK for integration with native iOS, Android, Windows 8 Universal, and Windows Phone 8 Silverlight applications, enabling implementation of various MobileFirst features such as connecting to the MobileFirst Server instance, calling Java and JavaScript adapters, implementing different authentication flows, certificate pinning, application authenticity protection, using push notifications, and more.
