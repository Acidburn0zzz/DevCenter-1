Welcome to the IBM MobileFirst Foundation Developer Center

We've created this space so that you could find the resources you need to develop Cordova (iOS, Android, Windows), web and native (iOS, Android, Windows) applications that are based on IBM MobileFirst Foundation. Detailed code tutorials, instructional video sessions and technical blog posts; all meant to get you started with ease. These applications can then be further enhanced by integrating with IBM Bluemix Mobile Services.

As a developer, this is the place for you to find an encompassing set of tutorials, enabling you to develop, test, manage, and secure your applications.

What you get

  • MobileFirst Server, the core of product, serving & protecting applications, breathing life into adapters, and integrating with other products.
  • MobileFirst Operations Console, a modern web UI for DevOps world with quick actions for creating applictions, live security configuration, sending push notifications, access to sample applications, adapters, and lots more.
  • MobileFirst CLI, enabling you to manage your artifacts in local and remote MobileFirst Servers, register applications as well as create adapters using simple & intuitive commands.
  • Client SDKs, available via the CocoaPods, Maven, NuGet and NPM package managers.
  • Adapter tooling based on Maven, to create the backend logic you always wanted.

What you can do

  • Integrate the MobileFirst SDK for Cordova, iOS, Android, Windows 8 Universal and Windows 10 UWP applications
  • Enrich the application with native functionality by using Cordova plug-ins and specialized APIs
  • Implement OAuth-based Security Checks and other authentication methods provided by the MobileFirst security framework
  • Connect to back-end systems to retrieve information using Java and JavaScript adapters
  • Update your (hybrid) application remotely without needing to re-submit to the app store
  • Block specific app versions from accessing the server (due to a security hole or alike)
  • Send targeted (or broadcast) push notifications
  • Glean user & application data using analytics APIs
  • Operate on data by using JSONStore and Cloudant

What is IBM MobileFirst Foundation?

IBM MobileFirst Foundation is a mobile development platform that provides development and deploy options both on the cloud (Bluemix) and on-premise (installed locally). MobileFirst Foundation enables you to build, enhance, and continuously deliver mobile apps efficiently and effectively.

With the MobileFirst Foundation, organizations can more effectively address the full lifecycle of mobile app development, delivery and on-going management. MobileFirst Foundation supports a wide range of development approaches from native to hybrid as well as web. Therefore, you can evaluate the best approach for each situation, according to skills, time and functionality, without being limited by a specific approach to mobile application development.

MobileFirst Foundation extend and integrate development, delivery and management capabilities by adding scalable data services, code scanning, quality assurance, and Bluemix mobile services.

The result is an integrated platform that accelerates the delivery of your mobile strategy with increased productivity and security, and provides users with a more engaging experience.

Comprehensive Security

Protect data on the device, APIs, and your brand from hackers and vulnerabilities. You don't need to be a security expert to get started right away with building secure apps. Employ best practices by using samples and templates, and quickly make and roll out changes with a configuration-based approach.

Backend Logic Engine

Add a rich back-end logic engine to power your apps. Write your app business logic in Java or Javascript. Expose your code as REST API endpoints, and document in Swagger to promote collaboration and simplify testing.

App Lifecycle Management

Adopt a more streamlined and repeatable process for incremental app updates. For example, skip the app store approval by making changes to your app while it's live, with no recompile needed.

Push and Offline Sync

Get essential back-end services that every app needs, pre-integrated in one easy-to-use SDK.

Bluemix Deployment

The Mobile Foundation service provisions and orchestrates the IBM MobileFirst Foundation platform in the Bluemix cloud environment. Your mobile servers are created in IBM Containers, which provide you with a reliable, scalable, and production-ready environment.

IBM Bluemix

IBM Bluemix is an open-standards, cloud platform for building, running, and managing applications.
With Bluemix, developers can focus on building excellent user experiences with flexible compute options, choice of DevOps tooling, and a powerful set of IBM and third-party APIs and services.
