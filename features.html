Product Features

Develop

Build better Enterprise apps. Speed up development & ship high quality apps.

Adapters

Quickly develop & run server side code to communicate with your backend systems securely. Add your business logic, and make it dynamically configurable so you can change app behavior without recompile.

Direct Update

Push over-the-air changes to front end web resources without app store review.

JSONStore

Encrypted on device storage & automated data sync for better app performance & offline access.

Crash Analytics

Make sure your app is stable, and that you can fix problems quickly.

App & API Performance Analytics

Stay ahead of problems, and keep your app responsive.

Secure

Protect your app, user data, and your APIs from hackers & vulnerabilities without without hiring mobile security experts.

Advanced Mobile Authentication

Easily add advanced mobile authentication to your apps, including step-up, multi-factor, SSO between apps & between back-ends and more.

App Authenticity

App fingerprinting to prevent access to your APIs from malicious apps.

Certificate pinning

Thwart man in the middle attacks.

Device Whitelisting & Blacklisting

Protect from device loss, theft, or misuse.

Manage

Remote Disable & Version Enforcement

Protect users from unsafe apps by disabling versions & forcing users to upgrade.

Dynamic Set-up of App Security & Back-End Configuration

Make changes in real-time to the app security configuration (eg. authentication method, max number of login attempts allowed) or backend configuration (eg Database URL), with zero downtime or recompile.

Engage

Mobile Analytics

Understand your users and how they use your app.

Push and SMS Notifications

Deliver messages to multiple platforms with a single API & send targeted messages from a web console.

Live Update

Change your app on the fly, roll new features out gradually, A/B test, and customize your app for different users.
