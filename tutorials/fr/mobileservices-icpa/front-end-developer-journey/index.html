Ajout du SDK MobileFirst
Les SDK client fournissent des API permettant aux développeurs d'implémenter diverses fonctionnalités Mobile Foundation, tels que les mécanismes d'authentification et de sécurité, les notifications push et SMS, les demandes de ressources, la collecte de données d'analyse, et plus encore.
