Overview

The Moving to production category is meant for all developers, whether aiming for native or hybrid applications.

In this category, the developer will learn how to move MobileFirst projects artifacts from the development environment to test and production environments, as well as any preparations needed before the move; configure and use Operational Analytics & Reports, distribute MobileFirst applications using the IBM MobileFirst Application Center and more.
