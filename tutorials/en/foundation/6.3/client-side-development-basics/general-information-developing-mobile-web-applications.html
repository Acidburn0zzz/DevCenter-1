Overview

In this tutorial a general development information for the Mobile Web environment will be presented.

Mobile Web applications run on smartphones and tablets.
You can optimize the application for Mobile Web by adding or changing HTML, CSS, and JavaScript files in the mobilewebapp environment folder.

Optimizing for Mobile Web

When developing a Mobile Web app, network performance is a highly important optimization to think about.
Network performance can be optimized by implementing the following two actions:

  • Use Minification and Concatenation to reduce the size and number of files that are used within the application.
  • Use HTML 5 Application Cache

For more information about Minification and Concatenation, see the "Optimizing MobileFirst applications" user documentation topic
