Relevant to:

  • Native Windows 8
  • Hybrid

Overview

Prerequisite: if you intend to use MobileFirst Studio to develop your application, first read the Setting up your development environment tutorial.

Before you start to develop, deploy, and test your Windows Phone applications, an additional step is required for setting up the Windows 8 and Windows Phone 8 development environment. This includes installing Visual Studio 2012.

Windows 8 and Windows Phone 8 development tools

Microsoft Visual Studio Express 2012 is an integrated development environment (IDE), which you can use to develop applications for either the Windows 8 or Windows Phone 8 platform.

Note: You must have a computer that runs Windows 8 OS to use it.

To download Visual Studio Express 2012 for Windows Phone, visit the Microsoft Visual Studio download website.
