Overview

The server-side development category is meant for all developers, whether they develop native or hybrid applications.

Server-side development provides the tools and functionality that are necessary to connect an application to MobileFirst Server or serve content to an application from an enterprise back end.

With these tutorials, developers learn how to develop server-side code by using MobileFirst adapters, invoke adapters, and implement various authentication mechanisms.
