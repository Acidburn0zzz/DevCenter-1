Follow the steps below to get quickly started with development using React Native on MobileFirst Platform Foundation.

Step 1: Setting up the React Native development environment

Follow the instructions provided in the Getting Started page in React Native docs, to set up your machine for React Native development.

This includes the setup required for Android & iOS.

Step 2: Adding the MobileFirst SDK to your React Native app

MobileFirst SDK for React Native is available as a React Native module from NPM.

Getting started with a new React Native project

  1. Create a new React Native project. 
          react-native init MyReactApp
  2. Add the MobileFirst SDK to your app. 
          cd MyReactApp
      npm install react-native-ibm-mobilefirst --save
  3. Link all native dependencies to your app. 
          react-native link

Step 3: Manual steps for Windows only

Perform the following manual steps if your development environment is on Windows.

  1. Copy the configuration file.

    Navigate to your React Native app's folder.

    Copy the mfpclient.properties file from <ReactNativeApp>\node_modules\react-native-ibm-mobilefirst\android\src\main\assets to <ReactNativeApp>\android\app\src\main\assets .

    You may have to create the assets folder if its not already present.

  2. Add internal dependencies.

    Navigate to your React Native app's folder.

    Append the contents of the <ReactNativeApp>\node_modules\react-native-ibm-first\scripts\bridge.settings to <ReactNativeApp>\android\settings.gradle.

Step 4: Calling a MobileFirst adapter

The WLResourceRequest class contains several methods for making calls to a MobileFirst adapter. See the API Reference for more details.

Make sure you go through the Developing Adapters section of the documentation to learn how to develop MobileFirst adapters.

To use the WLResourceRequest class in your React Native application, import the class reference in your application source.

In your App.js file (or any other source file where you want to call the adapter), import the WLResourceRequest class into your file. import {WLResourceRequest} from 'react-native-ibm-mobilefirst';

Below is an example code on how to call a MobileFirst adapter: 

  var balanceRequest = new WLResourceRequest(
 	 "adapters/ResourceAdapter/balance",  /* Adapter path */
 	 "GET", 									 /* Adapter method */
 	 5000,										 /* Timeout */
 	 "accessRestricted" );					 /* Scope protecting the adapter method */

     balanceRequest.send().then((response)=>{
       /* Adapter call was successful. Add your success flow code here */
     }).catch((error)=>{
       /* Adapter call failed. Add your failure flow code here */
     })
