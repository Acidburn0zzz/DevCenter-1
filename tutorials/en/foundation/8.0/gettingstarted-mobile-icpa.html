Build and Test

  • Download and install DevKit.
  • Start MobileFirst services from DevKit (CLI).
  • Download and install IBM Digital App Builder.
    • Low-code your first hybrid mobile application with IBM MobileFirst backend capabilities and test the app.
    • Export your MobileFirst application configurations (CLI).
  • Publish Mobile client application to AppStores or to IBM MobileFirst Application Center.

Deploy and Manage

  • Install ICPA.
  • Import your Mobile Foundation application configuration (CLI).
  • Start IBM Mobile Foundation Operations console & Analytics console and start managing your apps.
  • Publish Mobile client application to AppStores or to IBM MobileFirst Application Center.
