Mobile services in IBM Cloud Pak for Applications

Expedite the movement of your enterprise workloads to the cloud by using IBM Cloud Pak for Applications, which is an enterprise-grade environment that provides end-to-end capabilities to modernize your existing workloads or to build new cloud-native apps. Mobile services in IBM Cloud Pak for Applications enables you to create a mobile channel for your applications, solving some of the modern-day business challenges and fostering the fulfillment of some of the innovative business use cases.


Foundation Server

Mobile Foundation service simplifies building, managing, and updating your mobile applications with a comprehensive set of mobile application services.
Developers can build their application using the front-end framework and tools of choice, and easily add push, authentication, offline sync, and business logic to their apps with the Mobile software development kit (SDK) that is available across a wide range of supported mobile platforms.

Mobile Analytics

To keep your user engagement relevant and effective you must obtain insights into how your application is performing with users. Mobile Analytics service provides this capability with in-built visualizations (charts and tables). With very minimal instrumentation of your application, you can readily visualize, on the Mobile Foundation Analytics console insights such as User Onboarding & Retention Patterns, Device & OS Distribution, Backend Performance, Application Stability, and more.

Application Center

Application Center is a tool to make sharing mobile applications within an organization easier. You can use the Application Center as an enterprise application store. With the Application Center, you can target some mobile applications to particular groups of users within the company.
Developers can use the Application Center for collaborative application development.

Push Notifications

Notifications is the ability to push a message from the application backend running on the server to a mobile application user delivered on the device.
Push Notifications service provides a unified set of API methods to send either push or SMS notifications to iOS, Android, Windows 8.1 Universal, Windows 10 UWP and Cordova (iOS, Android) applications. The notifications are sent from the Mobile Server to the vendor (Apple, Google, Microsoft, SMS Gateways) infrastructure, and from there to the relevant devices.
