## 概述
{: #overview }
IBM MobileFirst Platform Foundation 修订包可应用于您的安装，以更新和升级 MobileFirst 组件。

应用更新或修订包包含以下两个步骤：

1. [运行 IBM Installation Manager](installation-manager)

2. [更新 MobileFirst Server](appserver-update)
