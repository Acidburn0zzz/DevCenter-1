フロントエンド開発者ジャーニー


AI 機能を持つ、モバイル・アプリケーション、Web アプリケーション、および PWA (Progressive Web App) マルチチャネル・アプリケーションを、ローコード手法で迅速に作成します


  • Cordova

  • Ionic

  • Android

  • iOS

Cordova アプリ作成ステップ

インストールおよび構成
使用可能なサービスおよびランタイム・フレームワークのエコシステムの成長を取り込みつつ、開発者がクラウド・アプリケーションの作成、デプロイ、および管理を迅速に行うことを可能にします。

MobileFirst SDK の追加
クライアント SDK は、開発者が認証とセキュリティー・メカニズム、プッシュと SMS の通知、リソース要求、分析データの収集、その他さまざまな Mobile Foundation フィーチャーを実装するための API のセットを提供します。

通知の送信
プロセス全体を開発者にまったく認識されないものにする一元的な通知メカニズムを通して、対象者をエンゲージします。

認証およびセキュリティーの追加
さまざまな保護レイヤーをアプリケーションに追加します。

測定およびエンゲージ
IBM Mobile Foundation を使用して通知を送信します。アプリケーション使用量をモニターし、洞察を得ます。
