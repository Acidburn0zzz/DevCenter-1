프론트 엔드 개발자 과정


코드를 적게 사용하는(low-code) 방식으로 AI 기능을 갖춘 모바일, 웹 및 PWA(Progressive Web App) 다중 채널 애플리케이션을 빠르게 작성합니다.


  • Cordova

  • Ionic

  • Android

  • iOS

Cordova 앱을 빌드하는 단계

설치 및 구성
개발자는 계속 증가하는 사용 가능한 서비스 및 런타임 프레임워크로 이루어진 에코시스템을 이용하면서, 자신의 클라우드 애플리케이션을 빠르게 빌드, 배치 및 관리할 수 있습니다.

MobileFirst SDK 추가
클라이언트 SDK는 개발자가 인증 및 보안 메커니즘, 푸시 및 SMS 알림, 리소스 요청, 분석 데이터 수집 등과 같은 다양한 Mobile Foundation 기능을 구현할 수 있는 API 세트를 제공합니다.

알림 전송
전체 프로세스를 개발자가 완전히 알 수 있도록 하는 통합 알림 메커니즘을 통해 사용자를 참여시킵니다.

인증 및 보안 추가
애플리케이션에 다양한 보호 계층을 추가합니다.

평가 및 참여
IBM Mobile Foundation에 대한 알림을 보냅니다. 애플리케이션 사용을 모니터하고 이에 대한 인사이트를 얻습니다.
